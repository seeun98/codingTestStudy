## [프로그래머스] 우유와 요거트가 담긴 장바구니
https://school.programmers.co.kr/learn/courses/30/lessons/62284

### solution
```sql
-- 코드를 입력하세요
select cart_id
from (select cart_id
from cart_products
where name = 'Milk'

intersect

select cart_id
from cart_products
where name = 'Yogurt')
order by cart_id



```
